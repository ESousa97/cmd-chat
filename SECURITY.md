# Política de Segurança

## Versões Suportadas

Atualmente as versões ativas estão garantidas a receber atualizações. Focamos no branch main.

## Notificar uma Vulnerabilidade
Se você descobrir um problema no `cmd-chat`, **por favor**, não os preencha como issues públicos do repositório.

Em vez disso, direcione informações sobre a descoberta para **[sneakybeaky18@gmail.com](mailto:sneakybeaky18@gmail.com)**. O tempo para resposta base atual é menor que 48 horas. Você receberá o devido crédito no `CHANGELOG` e todo suporte para patch release e atribuição de CVE.
